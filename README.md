# Retail Orders Analytics Pipeline

An end-to-end ETL and analytics project built using Python, Pandas, and SQL Server. The pipeline extracts retail order data from Kaggle, performs data cleaning and transformation, loads the processed data into SQL Server, and generates business insights through advanced SQL queries.

### Key Features

* Automated data ingestion from Kaggle datasets
* Data cleaning and preprocessing using Pandas
* Feature engineering for profit and sales analysis
* SQL Server integration for data storage
* Business analytics using joins, aggregations, CTEs, and window functions
* Revenue, profit, product, and regional performance analysis

### Tech Stack

* Python
* Pandas
* SQL Server
* SQLAlchemy
* Kaggle API
* PyODBC

### Business Insights Generated

* Top-selling products by revenue
* Region-wise product performance
* Month-over-month sales trends
* Category-wise peak sales periods
* Profit growth analysis by sub-category
* Overall revenue and profitability metrics
